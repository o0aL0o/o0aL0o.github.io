Man VS Alien

2D game
see details

Background

This game is a top view gun game. While the player is shooting a monster and killing them, he is getting an experience value in order to enhance the player levels. When upgrading his level, he could choose some new weapon or strengthening such as adding a heart or increase the attack. Also, the game is a level game. The player should pass each level and each level will increase with the number of passes.
